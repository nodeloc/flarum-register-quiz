A [Flarum](http://flarum.org) extension. Register Quiz

## Installation

Install with composer:

```sh
composer require xypp/flarum-register-quiz:"*"
```

## Updating

```sh
composer update xypp/flarum-register-quiz:"*"
php flarum migrate
php flarum cache:clear
```
